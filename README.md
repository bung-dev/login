## 📌 커밋 컨벤션

|------|-----------|------------------|
| ✨ | **feat** | 새로운 기능 추가 |
| 🐛 | **fix** | 버그 수정 |
| 📝 | **docs** | 문서 수정 (예: README 등, 코드 주석 제외) |
| 💄 | **style** | 코드 포맷팅 (공백, 세미콜론 등 기능 변화 없음) |
| ♻️ | **refactor** | 기능 변화 없이 코드 구조 개선 |
| ✅ | **test** | 테스트 코드 추가 또는 수정 |
| 🔧 | **chore** | 빌드, 패키지 설정 등 기능과 직접 관련 없는 작업 |
